Radar Estratégico Organizacional

El Radar Estratégico es una herramienta que permite visualizar el nivel de desarrollo de diferentes áreas clave dentro de una empresa. A través de este gráfico, se pueden identificar fortalezas y oportunidades de mejora en áreas como finanzas, cliente, negocio, entre otras.
